# v0.2.2-internal.3

## Scope

Internal refresh cut on top of the `v0.2.0` release line.

This release packages the Roboshare branding cleanup across the web app shell and public metadata assets after the initial smart-wallet foundation cut.

## Included Work

- Header logo now uses a dedicated Roboshare logo asset, with the subtitle rendered separately
- Dark mode header branding now uses a gold-accented logo variant
- Public favicon assets were rebuilt to match the header-style bolt geometry
- Public logo assets were simplified to a standalone bolt-plus-wordmark lockup
- Legacy Scaffold-ETH favicon, logo, and thumbnail assets were replaced with Roboshare branding

## Validation Expectations

- Header branding is consistent in light and dark mode
- Favicon renders cleanly inside its rounded background
- Public metadata references resolve the expected favicon assets
- Social preview thumbnail remains branded for Roboshare

## Notes

- This is an internal branding and asset refresh only; it does not change protocol or wallet-flow semantics
- The release branch remains `release/testnet-v0.2.0`
